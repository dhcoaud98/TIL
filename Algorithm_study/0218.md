```python
print(*temp[1:N+1])
'#1 1 2 2 2 0'

    print(temp[i], end = ' ')
'#1 1 2 2 2 0 '
```



2.  두 가지는 순서만 바뀐 것 처럼 보이지만 의미가 완전히 다르다. 

```python
(1) max_i = s
(2) s = max_i
```

 (1)은 max_i 에 s를 넣고, s는 변동이 없지만 (2)는 s에 max_i를 넣고, max_i는 변동이 없다.
