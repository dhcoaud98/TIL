### 4. Django_03(Model Form)

1. `forms.py`만들기

   ```python
   # forms.py
   
   from .models import Article
   
   class ArticleForm(forms.ModelForm):
   
       class Meta:
           model = Article
           fields = '__all__'
   ```

   * new 함수 삭제

   * `new.html`을 `create.html`로 변경

   * edit 함수 삭제
   * `edit.html`을 `update.html`로 변경!!

   def create(request):
