## Python 기본 문법





1. range의 범위를 거꾸로 만들고 싶을 때 

   ```python
   range(10, 0, -1)
   ```

   하면 10부터 -1씩 값이 나온다.  혹은

   ```python
   reversed(range(n))
   ```

   을 해도 거꾸로 진행된다. 



2. `print( ' ' )`를 할때 세로가 기본인데 가로로 나오게 만들고 싶을 때

   ```python
   print(i, end = ' ')
   ```

   

3. list 정렬

   ```python
   # 오름차순 정렬
   a.sort()
   
   # 내림차순 정렬 - reserve가 True일 때 내림차순
   a.sort(reverse = True)
   
   ```
